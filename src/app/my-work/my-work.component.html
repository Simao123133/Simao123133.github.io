Recipes Website

This website displays recipes, their nutritional information based on the inserted calories and nutrients per ingredient, has an authentication system and allows to leave messages to me. I used Django for the backend, with integration of SQLite3 to create the models. The frontend was made with Django Forms and Bootstrap. To host the website I used Heroku and to store files, Cloudinary.

Solar Boat Data Analysis

This is an app to view on or offline data incoming from a boat. The messages are received by MQTT and are displayed in customizable graphs using PyQt5 and PyQtGraph. It also allows calculations between signals.

Hydrofoil Control

This project is about the control of the hydrofoils of a solar powered boat. A lot of control, filtering and digital processing algorithms where developed to achieve stable flight. The code is written in MATLAB and Python and then translated to C++ in a Teensy 4.0.

Bayesian Network Classifier

This is a dataset classifier based on a Bayesian Network. Written in Java and has a small UI to enter datasets and view results.

Client Server Database ATM architecture

This is a client server database architecture simulating a working ATM. The ATM must communicate with the database to give access to the client and let them perform several operations. Was written in C and used stream and datagram sockets to communicate between the components. To avoid race conditions, mutexes were inserted on simultaneous write or delete actions. In addition, several threads were used to facilitate conversation and operations on the systems.

Boat Identifier

This code uses neural networks - feedforward and recurrent - to identify a model of a boat. Written in MATLAB and uses the deep learning toolbox and also a fuzzy toolbox.

Bubble Crush

This is a candy-crush like game with bubbles, whose goal is to combine 4 or more in rows or columns to eliminate all bubbles. The game was developed in C and was an introductory project to programming.
